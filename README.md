# Projeto DropBox Clone

Projeto desenvolvido como exemplo do Curso Completo de JavaScript na Udemy.com.

Com Firebase Realtime e Firebase Storage.

## Intalação

- npm install

## Dependências, css, etc, no arquivo bower.json

- npm install -g bower
- bower install
